# 教程
# 1.注册
首先打开 https://emos.lol 单击主页上的『立即体验』 此时如果没有意外的话就弹出登录框 如果你是新用户 请直接在用户名/密码输入框内输入你要创建的用户名密码 随后等待cloudflare的人机验证自动通过即可（当然 也可能会显示验证您是否为真人 点击框框验证即可）系统会自动提交您的信息 如果出现用户名或密码错误的提示 则代表您要创建的用户名已经被他人占用了😅 重新选择一个新用户名即可
# 2.登录emby服务器
在注册完成后 系统会自动跳转到用户面板 此时在面板上面已经显示了emby连接地址 如果没有显示请联系管理员👍 此时在你的emby连接软件里输入相关信息即可连接（不会的请看下图 这里以yamby为例）登录后就可以正常看片啦~
