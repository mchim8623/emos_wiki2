# 反代相关

本服欢迎各位大佬反代 也希望您将地址贡献出来

建议反代路径为 `/emby`

建议增加头部 `EMOS_PROXY_NAME` 值为你自己的称号 如 `@emos` 这样在客户端连接时也会显示出来

建议缓存 `/emby/Items/*/Images/*` 开头的图片地址

## 分流相关

本服域名较多 请根据实际情况配置

- `*.sharepoint.cn`
- `*.sharepoint.com`
- `*.emosstore.sbs`
